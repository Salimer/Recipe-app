# 📖 [Recipe App] <a name="about-project"></a>

> The Recipe app keeps track of all your foods, recipes, and ingredients. It allows you to save ingredients, create recipes, and generate a shopping list based on what you have and what you are missing from a recipe.

## 💻 Getting Started <a name="getting-started"></a>

To run this project, take a copy of the code and follow the instruction below.

### Setup

`Clone the project`

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
```

### Usage

Start server with:

```
rails server
```

Visit http://localhost:3000/ in your browser.

### Run tests

Install npm with:

Install rspec with:

```
bundle install
```

and

```
rails generate rspec:install
```

run the test with:

```
rspec spec
```

### Usage

- Run `rails server` to run the app.
